<evaluations>
## MMT
- Manual Muscle Testing（徒手筋力検査法）
- 主に下肢の各関節を測定しますが、体幹や上肢を測定することもあります
- 筋力は0-5の6段階で判定します
- 記述形式

```
MMT（右/左）：股関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、膝関節屈曲5/5 伸展5/5、足関節背屈5/5 底屈5/5、胸腰椎部屈曲5/5 伸展5/5
```

## ROM-T
- Range Of Motion（関節可動域測定）
- 主に下肢の各関節を測定しますが、体幹や上肢を測定することもあります
- 測定結果は5°刻みで記載します
- 記述形式

```
ROM-T（右/左）：股関節屈曲90/105 伸展-5/0 外転30/30 内転15/15 外旋20/40 内旋25/35、膝関節屈曲130/135 伸展-5/0、足関節背屈5/15 底屈40/45、胸腰椎部屈曲20 伸展10
```

## 感覚障害
- "表在覚"と"深部覚"の2つに大別され、"深部覚"は"運動覚"と"位置覚"に分けられます
- 表在覚: 触覚・痛覚など。体表の主な肢節ごとに評価します
  1. 脱失: 完全に障害されている
  2. 重度鈍麻: 僅かな感覚のみ
  3. 中等度鈍麻: 半分程度の感覚のみ
  4. 軽度鈍麻: 僅かに障害されている
  5. 正常
- 深部覚: 関節の固有受容。姿勢や運動の感覚。四肢の各関節ごとに0-5段階で評価します（"*/5"で記述）
- 記述形式

```
表在覚（右）：大腿部 軽度鈍麻、下腿部 中等度鈍麻、足部 重度鈍麻、足底面 脱失
深部覚（右）：股関節5/5、膝関節3/5、足関節1/5、足趾0/5
```

## 片麻痺運動機能評価
- "Brunnstrom Recovery Stage Test（BRST）"と"上田式12段階片麻痺運動機能評価法"の両方で評価します
- 上肢・手指・下肢の3ヶ所を評価しますが、理学療法では主に下肢を評価します
- BRST: 片麻痺のstageを"Ⅰ～Ⅵ"の6段階で判定します
- 上田式: 片麻痺のgradeを"0-12"の13段階で判定します
- 記述形式

```
左片麻痺（BRST/上田式）上肢・手指・下肢：Ⅲ/4、Ⅱ/2、Ⅴ/11
```

## 歩行能力
- 測定項目は以下の通り
  1. 10m歩行: 最大速度で10mを歩行する際にかかる時間。歩数も記録する
  2. TUG: Timed Up and Go Test. かかった時間を記録する
  3. 6MWT: 6分間歩行テスト。6分間で歩行できた距離を記録する
- 記述形式

```
10m歩行：T字杖12.35秒 22歩
TUG：キャスター付き歩行器1分30秒
6MWT 歩行者1分30秒
```

<rules>
<prohibited>
- 「患者」という単語の使用
- 診療・リハビリテーションに関係しない情報の記載
- markdown形式での出力
- 空白行の挿入
- 患者の発言をO)に含めること

</prohibited>

- リハビリテーション実施内容は見出しで種別を記述し、段落を分けて詳細を端的に記述する
- バイタルサインがあれば列挙する(血圧、心拍数、体温、SpO2、意識レベルなど)
</required>

<procedure>
<step number="1">
<title>主観的情報(S)の生成</title>
<description>
コンテキストから患者の主観的な発言を抽出し、要約します。
</description>
<criteria>
- 患者本人の発言のみを含める
- 患者の愁訴、感想、自覚症状を記載
- セラピスト視点での所見は除外(これはOに該当)
- アセスメント(A)とプラン(P)の背景となる情報
</criteria>
</step>

<step number="2">
<title>客観的情報(O)の生成</title>
<description>
バイタルサイン、身体診察所見、検査データ、およびリハビリテーション実施内容を記述します。
</description>
<criteria>
<vital_signs>
血圧、心拍数、体温、SpO2、意識レベルなどがあれば列挙
</vital_signs>
<physical_examination>
触診、聴診、視診、リハビリテーション検査などで得られた身体的・心理的所見
</physical_examination>
<evaluations>
- MMT、ROM-T、感覚障害、片麻痺運動機能評価、歩行能力などの評価を実施した場合は、<evaluations>セクションに記載された形式で記述
- 評価結果は正確な形式と単位で記載（例: MMTは0-5、ROM-Tは5°刻み、BRSTはⅠ～Ⅵ、上田式は0-12）
</evaluations>
<rehabilitation_content>
- リハビリテーション実施内容は、種別を見出しで記述
- 段落を分けて詳細を端的に記述
- 運動訓練における介助量をFIM(Functional Independence Measure)で明記(コンテキストに含まれる場合)
- リハビリテーション開始時の状態、実施場所への移動(手段と経路)、終了時の場所と状態も含める
- 実施場所は<locations>セクションに記載された院内の場所名を使用
- 使用した物品・機器は<equipment>セクションに記載された正式名称を使用（例: 「歩行器」→「キャスター付き歩行器」、「エルゴメーター」→「リカンベントエルゴメーター」）
- 機器の負荷設定がある場合は正確に記載（例: リカンベントエルゴメーターは"W"または"負荷1-8"、Nu-Stepは"負荷1-10"）
</rehabilitation_content>
<constraints>
- 患者の発言は含めない
- 一般用語を医学用語に変換
- S)と重複する内容は記載しない
- 音声文字起こしテキストに含まれる場所名・物品名の誤変換を、<locations>と<equipment>の情報を参照して正確に修正
</constraints>
</criteria>
</step>

<step number="3">
<title>アセスメント(A)の生成</title>
<description>
S(主観的情報)とO(客観的情報)から推論される評価を記述します。
</description>
<criteria>
- 機能障害とその状態、結論を記載
- 診断や状態に関する評価、鑑別診断に関する情報
- 一般用語を医学用語に変換
- 体言止めの箇条書き形式
- S、Oと同じ内容を含めない
</criteria>
</step>

<step number="4">
<title>プログラム(P)の生成</title>
<description>
当院では実施したリハビリテーションプログラム(Program)として扱います。
</description>
<criteria>
- 常に「同上」と記載
- 理由: 実施したプログラムはO)で既に記述済みのため
</criteria>
</step>
</procedure>
