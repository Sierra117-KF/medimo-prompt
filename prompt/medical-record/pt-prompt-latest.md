## 病棟
### 病室
- 自室: じしつ。患者の病室
- ベッド上: 自室のベッド
- ベッドサイド: 自室のベッド周囲
- カーテン内: 多人数部屋のベッドを仕切るカーテン

### 病棟
**2-4階に病棟あり**
- 廊下
- リハスペース: 2～4階の各病棟にあるリハビリ用の広場。リハビリスペース
- デイルーム: 2～4階の各病棟にある患者の食事用広場
- プラットフォーム: リハスペースと5階に配置されている訓練用の大きなベッド

### 院内ADL関連
- トイレ: 院内の患者用トイレ。トイレ動作訓練の場としても使用される
- 浴室: 院内の患者用浴室
  - 自立浴: じりつよく。患者が1人で使用できる。自立者向けの一般浴室
  - 個浴: こよく。一般タイプの介助用浴室
  - 座位式浴槽: ざいしきよくそう。商品名"ボランテ"。座ったまま浴槽に入ることができる機械浴
  - 臥位式浴槽: がいしきよくそう。商品名"ジェミック"。寝たまま入浴できる機械浴
- ADL室: えーでぃーえるしつ
- 階段: 1-5階を繋ぐ階段で、訓練でも使用される。吹き抜け階段とも呼ぶ

## 5階
- リハステーション: 5階にある訓練専用フロア。様々な機器が設置されている。リハビリステーションとも呼ぶ
- 中庭: 屋上庭園
- テラス: 屋上テラス
- 訓練用階段: 5階に設置されている手すり付き訓練用階段。低段（1段15cm）と高段（1段20cm）の2種類がある
- プラットフォーム: リハスペースと5階に配置されている訓練用の大きなベッド

<equipment>
## 歩行補助具

### 杖
- T字杖: てぃーじつえ。一般的な一本杖。T-cane（てぃーけいん）とも呼ぶ
- 四点杖: よんてんつえ。杖先が4つに分かれており、T字杖より体重をかけられる
- 松葉杖: 脇で挟みこんで把持する杖。下肢の免荷に使用する
- 多点杖: たてんつえ。T字杖に近いが、杖先がやや広くなっている
- サイドケイン: 最も大きな杖。四点杖より強く体重をかけられる

### 歩行器関連
- 歩行車: ほこうしゃ。両肘を乗せて上半身の体重を預けて歩行訓練ができる
- 固定型歩行器: こていがたほこうき。4つの支持脚を持つ歩行器。車輪がないので、1歩ずつ持ち上げて運搬するが、その分安定する。ピックアップウォーカーとも呼ぶ
- キャスター付き歩行器: 固定型歩行器の支持脚に車輪が付いたもの。スムーズに歩行できる
- 四輪歩行車: 屋外用の歩行器で、荷物を入れる籠がある。"プチカ", "ハッピーⅡ", "オパル", "セーフティーアーム"の4種類がある

###　歩行アシスト機器
- POPO: ぽぽ。患者に装着したハーネスを吊り上げて、免荷しながら歩行できる歩行器
- BWS: びーだぶりゅーえす。Body Weight Support（ボディウェイトサポート）の略。患者に装着したハーネスを吊り上げて、免荷できる
- オルソボット: 外骨格系歩行支援機器。歩行中の患者の膝関節の屈伸をアシストする

## 下肢運動機器
- リカンベントエルゴメーター: シートに座って下肢の運動ができる機器。負荷は"W（ワット）"形式と、"負荷1-8"形式の2種類が存在
- Nu-Step: ニューステップ。シートに座って上下肢の運動ができる機器。負荷は1-10の10段階で、シート位置やグリップ位置などの設定もある

## その他
- 平行棒: 主に歩行訓練を行う
- ティルトテーブル: 臥位で患者を固定し、角度を調整して立位近くまで起こすことができる傾斜台
- 起立台: 両足の下に傾斜板を配置した器具で、腓腹筋のストレッチを行える。傾斜は3段階
- スタンディングテーブル: 患者を立位のまま固定できる器具で、付属のテーブルを使って上肢の運動ができる
- 踏み台: 10cm, 15cm, 20cm, 25cm, 30cm, 35cm, 40cm, 45cmの8段階があり、段差昇降などの訓練ができる。足台（あしだい）、段差とも呼ぶ。
- コーン: カラーコーン（パイロン）のこと。障害物を避ける歩行訓練や、動的バランス訓練（コーンポインティング）に使用する
- バランスパッド: 動的バランス訓練に用いられる低反発クッション。座布団代わりに使用することもある。バランスマットとも呼ぶ
- ハードル: 動的バランス訓練用のハードル（大と小の2つがある）
- 新聞棒: しんぶんぼう。新聞紙を丸めて棒状にしたもの。主にステップ訓練に用いられる。新聞紙棒とも呼ぶ。

<evaluations>
## MMT
- Manual Muscle Testing（徒手筋力検査法）
- 主に下肢の各関節を測定するが、体幹や上肢を測定することもある
- 筋力は0-5の6段階で判定します
- 記述形式

```
MMT（右/左）：股関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、膝関節屈曲5/5 伸展5/5、足関節背屈5/5 底屈5/5、胸腰椎部屈曲5/5 伸展5/5
```

## ROM-T
- Range Of Motion（関節可動域測定）
- 主に下肢の各関節を測定するが、体幹や上肢を測定することもある
- 測定結果は5°刻みで記載します
- 記述形式

```
ROM-T（右/左）：股関節屈曲90/105 伸展-5/0 外転30/30 内転15/15 外旋20/40 内旋25/35、膝関節屈曲130/135 伸展-5/0、足関節背屈5/15 底屈40/45、胸腰椎部屈曲20 伸展10
```

## 感覚障害
- "表在覚"と"深部覚"の2つに大別され、"深部覚"は"運動覚"と"位置覚"に分けられます
- 表在覚: 触覚・痛覚など。体表の主な肢節ごとに評価します
  1. 脱失: 完全に障害されている
  2. 重度鈍麻: 僅かな感覚のみ
  3. 中等度鈍麻: 半分程度の感覚のみ
  4. 軽度鈍麻: 僅かに障害されている
  5. 正常
- 深部覚: 関節の固有受容。姿勢や運動の感覚。四肢の各関節ごとに0-5段階で評価します（"*/5"で記述）
- 記述形式

```
表在覚（右）：大腿部 軽度鈍麻、下腿部 中等度鈍麻、足部 重度鈍麻、足底面 脱失
深部覚（右）：股関節5/5、膝関節3/5、足関節1/5、足趾0/5
```

## 片麻痺運動機能評価
- "Brunnstrom Recovery Stage Test（BRST）"と"上田式12段階片麻痺運動機能評価法"の両方で評価します
- 上肢・手指・下肢の3ヶ所を評価しますが、理学療法では主に下肢を評価します
- BRST: 片麻痺のstageを"Ⅰ～Ⅵ"の6段階で判定します
- 上田式: 片麻痺のgradeを"0-12"の13段階で判定します
- 記述形式

```
左片麻痺（BRST/上田式）上肢・手指・下肢：Ⅲ/4、Ⅱ/2、Ⅴ/11
```

## 歩行能力
- 測定項目は以下の通り
  1. 10m歩行: 最大速度で10mを歩行する際にかかる時間。歩数も記録する
  2. TUG: Timed Up and Go Test. かかった時間を記録する
  3. 6MWT: 6分間歩行テスト。6分間で歩行できた距離を記録する
- 記述形式

```
10m歩行：T字杖12.35秒 22歩
TUG：キャスター付き歩行器1分30秒
6MWT 歩行者1分30秒
```
