## 歩行補助具

### 杖
- T字杖: てぃーじつえ。一般的な一本杖。T-cane（てぃーけいん）とも呼ぶ
- 四点杖: よんてんつえ。杖先が4つに分かれており、T字杖より体重をかけられる
- 松葉杖: 脇で挟みこんで把持する杖。下肢の免荷に使用する
- 多点杖: たてんつえ。T字杖に近いが、杖先がやや広くなっている
- サイドケイン: 最も大きな杖。四点杖より強く体重をかけられる

### 歩行器関連
- 歩行車: ほこうしゃ。両肘を乗せて上半身の体重を預けて歩行訓練ができる
- 固定型歩行器: こていがたほこうき。4つの支持脚を持つ歩行器。車輪がないので、1歩ずつ持ち上げて運搬するが、その分安定する。ピックアップウォーカーとも呼ぶ
- キャスター付き歩行器: 固定型歩行器の支持脚に車輪が付いたもの。スムーズに歩行できる
- 四輪歩行車: 屋外用の歩行器で、荷物を入れる籠がある。"プチカ", "ハッピーⅡ", "オパル", "セーフティーアーム"の4種類がある

## 運動機器
- AR2: えーあーるつー。上肢訓練用ロボット支援機器
- エスパージ: 低周波治療機器。電気療法による筋収縮を運動に取り入れる
- リカンベントエルゴメーター: シートに座って下肢の運動ができる機器。負荷は"W（ワット）"形式と、"負荷1-8"形式の2種類が存在
- Nu-Step: ニューステップ。シートに座って上下肢の運動ができる機器。負荷は1-10の10段階で、シート位置やグリップ位置などの設定もある

## その他
- 平行棒: 主に歩行訓練を行う
- ティルトテーブル: 臥位で患者を固定し、角度を調整して立位近くまで起こすことができる傾斜台
- 起立台: 両足の下に傾斜板を配置した器具で、腓腹筋のストレッチを行える。傾斜は3段階
- スタンディングテーブル: 患者を立位のまま固定できる器具で、付属のテーブルを使って上肢の運動ができる
- 踏み台: 10cm, 15cm, 20cm, 25cm, 30cm, 35cm, 40cm, 45cmの8段階があり、段差昇降などの訓練ができる。足台（あしだい）、段差とも呼ぶ。
- コーン: カラーコーン（パイロン）のこと。障害物を避ける歩行訓練や、動的バランス訓練（コーンポインティング）に使用する
- バランスパッド: 動的バランス訓練に用いられる低反発クッション。座布団代わりに使用することもある。バランスマットとも呼ぶ
- ハードル: 動的バランス訓練用のハードル（大と小の2つがある）
- 新聞棒: しんぶんぼう。新聞紙を丸めて棒状にしたもの。主にステップ訓練に用いられる。新聞紙棒とも呼ぶ。
- サンディング: タオルやブロックを使い、上肢を運動させること
- 輪投げ: 上肢訓練用器具。輪入れとも呼ぶ
- ビー玉: 上肢訓練用に使用
- アクリルコーン: 上肢訓練用のコーン型器具
- 重錘: 訓練用の重り
- ペグ: 上肢機能訓練用の木製器具
- デザインペグ: 上肢機能＋認知機能訓練用の木製器具
- 上衣: じょうい。上着、上半身の衣類
- 下衣: かい。ズボン、下半身の衣類

<evaluations>
## MMT
- えむえむてぃー
- Manual Muscle Testing（徒手筋力検査法）
- 主に上肢の各関節を測定するが、体幹を測定することもある
- 筋力は0-5の6段階で判定する
- 記述形式

```
MMT（右/左）：肩関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、肘関節屈曲5/5 伸展5/5、前腕回内5/5 回外5/5、手関節背屈5/5 掌屈5/5、MP関節屈曲5/5 伸展5/5、PIP関節屈曲5/5 伸展5/5、DIP関節屈曲5/5 伸展5/5、胸腰椎部屈曲5/5 伸展5/5
```

## ROM-T
- あーるおーえむ
- Range Of Motion（関節可動域測定）
- 主に上肢の各関節を測定するが、体幹を測定することもある
- 測定結果は5°刻みで記載する
- 記述形式

```
ROM-T（右/左）：肩関節屈曲90/105 伸展10/20 外転60/110 内転10/15 外旋20/40 内旋25/35、肘関節屈曲120/135 伸展0/-5、前腕回内60/90 回外60/90、手関節背屈5/15 掌屈30/50、MP関節屈曲40/90 伸展10/20、PIP関節屈曲60/90 伸展0/0、DIP関節屈曲30/45 伸展0/0、胸腰椎部屈曲20 伸展10
```

## 感覚障害
- "表在覚"と"深部覚"の2つに大別され、"深部覚"は"運動覚"と"位置覚"に分けられる
- 表在覚: 触覚・痛覚など。体表の主な肢節ごとに評価する
  1. 脱失: 完全に障害されている
  2. 重度鈍麻: 僅かな感覚のみ
  3. 中等度鈍麻: 半分程度の感覚のみ
  4. 軽度鈍麻: 僅かに障害されている
  5. 正常
- 深部覚: 関節の固有受容。姿勢や運動の感覚。四肢の各関節ごとに0-5段階で評価します（"*/5"で記述）
- 記述形式

```
表在覚（右）：上腕部 軽度鈍麻、前腕部 中等度鈍麻、手部 重度鈍麻、手掌面 脱失
深部覚（右）：肩関節5/5、肘関節3/5、手関節1/5、手指0/5
```

## 片麻痺運動機能評価
- "Brunnstrom Recovery Stage Test（BRST）"と"上田式12段階片麻痺運動機能評価法"の両方で評価する
- 上肢・手指・下肢の3ヶ所を評価するが、作業療法では主に上肢・手指を評価する
- BRST: びーあーるえすてぃー。片麻痺のstageを"Ⅰ～Ⅵ"の6段階で判定する
- 上田式: 片麻痺のgradeを"0-12"の13段階で判定する
- 記述形式

```
左片麻痺（BRST/上田式）上肢・手指・下肢：Ⅲ/4、Ⅱ/2、Ⅴ/11
```

## 上肢機能評価
- 握力: 握力の強さを"kg"で判定。小数点第一位まで記載
- BBT: びーびーてぃー。Box and Block Testの略。1分間でブロックを運搬できる個数を測定
- ARAT: あらーと。Action Research Arm Testの略。上肢機能評価。57点満点
- STEF: すてふ。Simple Test for Evaluating Hand Functionの略。上肢機能評価。100点満点

- 記述形式

```
握力（右/左；kg）：12.5/22.0
BBT（右/左；個）：60/80
ARAT（右/左；点）：40/57
STEF（右/左；点）：80/100
```

## ADL訓練
- 調理訓練: ADL室で実際に調理してもらい、動作能力の評価を行う。主に野菜炒めを作る

## ADL評価
- FIM: ふぃむ。Functional Independence Measureの略。1-7点の7段階で判定
  - 音声文字起こしでは「リム」「ヒム」などに誤変換されやすい
  - FIM1: 全介助 
  - FIM2: 最大介助（75%程度解除）
  - FIM3: 中等度介助（50%程度解除）
  - FIM4: 軽介助（25%程度の介助）
  - FIM5: 見守り 
  - FIM6: 修正自立（補助具を使用すれば自立）
  - FIM7: 自立（補助具なしで自立）

## 歩行形態
**歩行する上での様式や状態を指す。自立度は別途FIMで判定する**
- 独歩: どっぽ。歩行補助具を使用せずに歩くこと
- 手引き歩行: てびきほこう。セラピストが手を繋いで介助して歩くこと（独歩を介助している状態）
- T字杖歩行: てぃーじつえほこう。T字杖を使用した歩行
- キャスター付き歩行器歩行: キャスター付き歩行器を使用した歩行
- 固定型歩行器歩行: 固定型歩行器を使用した歩行
- 歩行者歩行: 歩行者を使用した歩行

## その他
- HDS-R: えいちでぃーえすあーる。改訂長谷川式簡易知能評価スケール（Revised Hasegawa Dementia Scale）の略称。30点満点
