# 理学療法士がカルテに記載する評価

## MMT
- Manual Muscle Testing（徒手筋力検査法）
- 主に下肢の各関節を測定しますが、体幹や上肢を測定することもあります
- 筋力は0-5の6段階で判定します
- 記述形式

```
MMT（右/左）：股関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、膝関節屈曲5/5 伸展5/5、足関節背屈5/5 底屈5/5、胸腰椎部屈曲5/5 伸展5/5
```

## ROM-T
- Range Of Motion（関節可動域測定）
- 主に下肢の各関節を測定しますが、体幹や上肢を測定することもあります
- 測定結果は5°刻みで記載します
- 記述形式

```
ROM-T（右/左）：股関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、膝関節屈曲5/5 伸展5/5、足関節背屈5/5 底屈5/5、胸腰椎部屈曲5/5 伸展5/5
```

## 感覚障害
- "表在覚"と"深部覚"の2つに大別され、"深部覚"は"運動覚"と"位置覚"に分けられます
- 表在覚: 触覚・痛覚など。体表の主な肢節ごとに評価します
  1. 脱失: 完全に障害されている
  2. 重度鈍麻: 僅かな感覚のみ
  3. 中等度鈍麻: 半分程度の感覚のみ
  4. 軽度鈍麻: 僅かに障害されている
  5. 正常
- 深部覚: 関節の固有受容。姿勢や運動の感覚。四肢の各関節ごとに0-5段階で評価します（"*/5"で記述）
- 記述形式

```
表在覚（右）：大腿部 軽度鈍麻、下腿部 中等度鈍麻、足部 重度鈍麻、足底面 脱失
深部覚（右）：股関節5/5、膝関節3/5、足関節1/5、足趾0/5
```

## 片麻痺運動機能評価
- "Brunnstrom Recovery Stage Test（BRST）"と"上田式12段階片麻痺運動機能評価法"の両方で評価します
- 上肢・手指・下肢の3ヶ所を評価しますが、理学療法では主に下肢を評価します
- BRST: 片麻痺のstageを"Ⅰ～Ⅵ"の6段階で判定します
- 上田式: 片麻痺のgradeを"0-12"の13段階で判定します
- 記述形式

```
左片麻痺（BRST/上田式）上肢・手指・下肢：Ⅲ/4、Ⅱ/2、Ⅴ/11
```

## 歩行能力
- 測定項目は以下の通り
  1. 10m歩行: 最大速度で10mを歩行する際にかかる時間。歩数も記録する
  2. TUG: Timed Up and Go Test. かかった時間を記録する
  3. 6MWT: 6分間歩行テスト。6分間で歩行できた距離を記録する
- 記述形式

```
10m歩行：T字杖12.35秒 22歩
TUG：キャスター付き歩行器1分30秒
6MWT 歩行者1分30秒
```
