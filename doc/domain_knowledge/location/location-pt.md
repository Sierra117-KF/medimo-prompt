# 理学療法を行う場所

## 病棟

### 病室
- 自室: じしつ。患者の病室。
- ベッド上: 自室の患者用ベッド。
- ベッドサイド: 自室の患者ベッド周囲。
- カーテン内: 多人数部屋で患者のベッドを仕切るカーテン。

### 病棟
**2-4階に病棟あり**
- 廊下
- リハスペース: 2～4階の各病棟にあるリハビリ用の広場。リハビリスペース。
- デイルーム: 2～4階の各病棟にある患者の食事用広場。
- プラットフォーム: リハスペースと5階に配置されている訓練用の大きなベッド。

### 院内ADL関連
- トイレ: 院内の患者用トイレ。トイレ動作訓練の場としても使用される。
- 浴室: 院内の患者用浴室。
  - 自立欲: じりつよく。患者が1人で使用できる。自立者向けの一般浴室。
  - 個浴: こよく。一般タイプの介助用浴室。
  - 座位式浴槽: ざいしきよくそう。商品名"ボランテ"。座ったまま浴槽に入ることができる機械浴。
  - 臥位式浴槽: がいしきよくそう。商品名"ジェミック"。寝たまま入浴できる機械浴。
- ADL室: えーでぃーえるしつ。
- 階段（もしくは"吹き抜け階段"）: 1-5階を繋ぐ階段で、患者の訓練でも使用される。

### 5階
- リハステーション: 5階にある訓練専用フロア。様々な機器が設置されている。リハビリステーション。
- 中庭: 屋上庭園
- テラス: 屋上テラス
- 訓練用階段: 5階に設置されている手すり付き訓練用階段。低段（1段15cm）と高段（1段20cm）の2種類がある。
- プラットフォーム: リハスペースと5階に配置されている訓練用の大きなベッド。
