# 理学療法士がカルテに記載する評価

## MMT
- えむえむてぃー
- Manual Muscle Testing（徒手筋力検査法）
- 主に下肢の各関節を測定しますが、体幹や上肢を測定することもある
- 筋力は0-5の6段階で判定する
- 記述形式

```
MMT（右/左）：股関節屈曲5/5 伸展5/5 外転5/5 内転5/5 外旋5/5 内旋5/5、膝関節屈曲5/5 伸展5/5、足関節背屈5/5 底屈5/5、胸腰椎部屈曲5/5 伸展5/5
```

## ROM-T
- あーるおーえむ
- Range Of Motion（関節可動域測定）
- 主に下肢の各関節を測定する、体幹や上肢を測定することもある
- 測定結果は5°刻みで記載する
- 記述形式

```
ROM-T（右/左）：股関節屈曲90/105 伸展-5/0 外転30/30 内転15/15 外旋20/40 内旋25/35、膝関節屈曲130/135 伸展-5/0、足関節背屈5/15 底屈40/45、胸腰椎部屈曲20 伸展10
```

## 感覚障害
- "表在覚"と"深部覚"の2つに大別され、"深部覚"は"運動覚"と"位置覚"に分けられる
- 表在覚: 触覚・痛覚など。体表の主な肢節ごとに評価する
  1. 脱失: 完全に障害されている
  2. 重度鈍麻: 僅かな感覚のみ
  3. 中等度鈍麻: 半分程度の感覚のみ
  4. 軽度鈍麻: 僅かに障害されている
  5. 正常
- 深部覚: 関節の固有受容。姿勢や運動の感覚。四肢の各関節ごとに0-5段階で評価します（"*/5"で記述）
- 記述形式

```
表在覚（右）：大腿部 軽度鈍麻、下腿部 中等度鈍麻、足部 重度鈍麻、足底面 脱失
深部覚（右）：股関節5/5、膝関節3/5、足関節1/5、足趾0/5
```

## 片麻痺運動機能評価
- "Brunnstrom Recovery Stage Test（BRST）"と"上田式12段階片麻痺運動機能評価法"の両方で評価する
- 上肢・手指・下肢の3ヶ所を評価するが、理学療法では主に下肢を評価する
- BRST: びーあーるえすてぃー。片麻痺のstageを"Ⅰ～Ⅵ"の6段階で判定する
- 上田式: 片麻痺のgradeを"0-12"の13段階で判定する
- 記述形式

```
左片麻痺（BRST/上田式）上肢・手指・下肢：Ⅲ/4、Ⅱ/2、Ⅴ/11
```

## 歩行能力評価
- 測定項目は以下の通り
  1. 10m歩行: 最大速度で10mを歩行する際にかかる時間。歩数も記録する
  2. TUG: てぃーゆーじー。Timed Up and Go Test. かかった時間を記録する
  3. 6MWT: 6分間歩行テスト。6分間で歩行できた距離を記録する
  4. FSST: えふえすえすてぃー。Four Square Step Testの略。動的バランステスト。
- 記述形式

```
10m歩行：T字杖12.35秒 22歩
TUG：キャスター付き歩行器1分30秒
6MWT：歩行車1分30秒
FSST：15.3秒
```

## ADL評価
- FIM: ふぃむ。Functional Independence Measureの略。1-7点の7段階で判定
  - FIM1: 全介助 
  - FIM2: 最大介助（75%程度解除）
  - FIM3: 中等度介助（50%程度解除）
  - FIM4: 軽介助（25%程度の介助）
  - FIM5: 見守り 
  - FIM6: 修正自立（補助具を使用すれば自立）
  - FIM7: 自立（補助具なしで自立）

## 歩行形態
**歩行する上での様式や状態を指す。自立度は別途FIMで判定する**
- 独歩: どっぽ。歩行補助具を使用せずに歩くこと
- 手引き歩行: てびきほこう。セラピストが手を繋いで介助して歩くこと（独歩を介助している状態）
- T字杖歩行: てぃーじつえほこう。T字杖を使用した歩行
- キャスター付き歩行器歩行: キャスター付き歩行器を使用した歩行
- 固定型歩行器歩行: 固定型歩行器を使用した歩行
- 歩行者歩行: 歩行者を使用した歩行

## その他
