# 言語聴覚士がカルテに記載する評価

## 認知機能検査
- HDS-R: えいちでぃーえすあーる。改訂長谷川式簡易知能評価スケール（Revised Hasegawa Dementia Scale）の略称。30点満点
- レーブン色彩マトリックス検査: 非言語性の知能（類推能力）を検査する。36点満点
- リバーミード行動記憶検査: 記憶障害の程度を検査する。(標準プロフィール24点満点、スクリーニング12点満点)
- 三宅式記銘力検査: みやけしききめいりょくけんさ。記憶力を検査する

## 高次脳機能検査
- CAT視覚性抹消課題: きゃっとしかくせいまっしょうかだい。視覚的な注意力（探索能力）を検査する(正答率%、所要秒)
- TMT-JA : てぃーえむてぃーじぇいえー。Trail Making Test - Part Aの略。視覚的な注意力や処理速度を検査する(所要秒)
- TMT-JA : てぃーえむてぃーじぇいびー。Trail Making Test - Part Bの略。注意の切り替え能力を検査する(所要秒)
- 立方体模写: りっぽうたいもしゃ。視空間認知や構成能力を検査する
- 線分抹消試験: せんぶんまっしょうしけん。半側空間無視の有無や程度を検査する
- 線分二等分試験: せんぶんにとうぶんしけん。半側空間無視の有無や程度を検査する
- BADS: ばっど、ばっづ。Behavioural Assessment of the Dysexecutive Syndromeの略。実行機能の検査(総プロフィール得点24点満点、全体区分)

## 音声・発話機能検査
- 発話明瞭度: 聞き手にとってどれくらい明瞭に理解できるかを評価する


## 嚥下機能検査
- 摂食嚥下グレード: 摂食嚥下機能の状態を、段階（グレード、レベル）で判定する(Gr.,Lv. 重症度)
- 反復唾液飲みテスト: Repetitive Saliva Swallowing Test（RSST: あーるえすえすてぃー）。嚥下機能を簡易的に検査する（回/30秒）
- 改訂水飲みテスト: Modified Water Swallowing Test（MWST: えむだぶりゅーえすてぃー）。嚥下機能を評価する(評点)



## 知能検査
- WAIS-Ⅳ: うぇいすあーる。ウェクスラー成人知能検査 第4版の略。全検査（FSIQ）、言語理解（VCI）、知覚推理（PRI）、ワーキングメモリー（WMI）、処理速度（PSI）
- WMS-R: ウェクスラー記憶検査。記憶能力を総合的に検査する（言語性記憶、視覚性記憶、一般的記憶、注意/集中力、遅延再生）
- コース立方体組み合わせテスト: 視空間構成能力や非言語性の知能を検査する(IQ)
