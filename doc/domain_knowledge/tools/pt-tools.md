# 理学療法で使用する物品・器具・機器

## 歩行補助具

### 杖
- T字杖: てぃーじつえ。一般的な一本杖。T-cane（てぃーけいん）とも呼ぶ
- 四点杖: よんてんつえ。杖先が4つに分かれており、T字杖より体重をかけられる
- 松葉杖: 脇で挟みこんで把持する杖。下肢の免荷に使用する
- 多点杖: たてんつえ。T字杖に近いが、杖先がやや広くなっている
- サイドケイン: 最も大きな杖。四点杖より強く体重をかけられる

### 歩行器関連
- 歩行車: ほこうしゃ。両肘を乗せて上半身の体重を預けて歩行訓練ができる
- 固定型歩行器: こていがたほこうき。4つの支持脚を持つ歩行器。車輪がないので、1歩ずつ持ち上げて運搬するが、その分安定する。ピックアップウォーカーとも呼ぶ
- キャスター付き歩行器: 固定型歩行器の支持脚に車輪が付いたもの。スムーズに歩行できる
- 四輪歩行車: 屋外用の歩行器で、荷物を入れる籠がある。"プチカ", "ハッピーⅡ", "オパル", "セーフティーアーム"の4種類がある

###　歩行アシスト機器
- POPO: ぽぽ。患者に装着したハーネスを吊り上げて、免荷しながら歩行できる歩行器
- BWS: びーだぶりゅーえす。Body Weight Support（ボディウェイトサポート）の略。患者に装着したハーネスを吊り上げて、免荷できる
- オルソボット: 外骨格系歩行支援機器。歩行中の膝関節の屈伸をアシストする

## 下肢運動機器
- リカンベントエルゴメーター: シートに座って下肢の運動ができる機器。負荷は"W（ワット）"形式と、"負荷1-8"形式の2種類が存在
- Nu-Step: ニューステップ。シートに座って上下肢の運動ができる機器。負荷は1-10の10段階で、シート位置やグリップ位置などの設定もある

## 下肢装具
- 長下肢装具: ちょうかしそうぐ。金属支柱付きの長下肢装具。LLB（Long Leg Brace: えるえるびー）、KAFO（Knee Ankle Foot Orthosis: けーえーえふおー）とも呼ぶ
- 体幹ベルト付き長下肢装具: 左右両側の長下肢装具を体幹ベルトで連結したもので、股継手も付属している
- 短下肢装具: たんかしそうぐ
- SLB: えすえるびー。金属支柱付き短下肢装具
- シューホーン: プラスチック型の標準的な短下肢装具。シューホーンブレースとも呼ぶ
- タマラック: 足継手付きプラスチック短下肢装具
- オルトップ: 小型プラスチック短下肢装具。オルトップLH（えるえいち）、オルトップLH+（えるえいちぷらす）の2種類がある
- UD-Flex: ゆーでぃーふれっくす。小型プラスチック短下肢装具
- GS-D: じーえすでぃー。油圧ダンパー足継手付き短下肢装具

## その他
- 平行棒: 主に歩行訓練を行う
- ティルトテーブル: 臥位で患者を固定し、角度を調整して立位近くまで起こすことができる傾斜台
- 起立台: 両足の下に傾斜板を配置した器具で、腓腹筋のストレッチを行える。傾斜は3段階
- スタンディングテーブル: 患者を立位のまま固定できる器具で、付属のテーブルを使って上肢の運動ができる
- 踏み台: 10cm, 15cm, 20cm, 25cm, 30cm, 35cm, 40cm, 45cmの8段階があり、段差昇降などの訓練ができる。足台（あしだい）、段差とも呼ぶ。
- コーン: カラーコーン（パイロン）のこと。障害物を避ける歩行訓練や、動的バランス訓練（コーンポインティング）に使用する
- バランスパッド: 動的バランス訓練に用いられる低反発クッション。座布団代わりに使用することもある。バランスマットとも呼ぶ
- ハードル: 動的バランス訓練用のハードル（大と小の2つがある）
- 新聞棒: しんぶんぼう。新聞紙を丸めて棒状にしたもの。主にステップ訓練に用いられる。新聞紙棒とも呼ぶ。
