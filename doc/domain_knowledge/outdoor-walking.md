# 屋外歩行訓練に関するドメイン知識

## 概要
- 屋外歩行（おくがいほこう）: リハビリテーション実施中、院外での歩行訓練を行うことを指す
- 理学療法、作業療法では主に運動療法として行う
- 言語聴覚療法においては、高次脳機能の評価・訓練を目的に行う場合もある

## 屋外歩行コース
- 数字: 病院周囲の各交差点に、1から順に数字を割り当てている
- Aコース: えーこーす。病院敷地内を周回するコース
- ベンチ: 規定コース内には、座って休憩できるベンチが複数配置されている

## 記述形式
```
・屋外歩行
　T字杖歩行　FIM4　Aコース2周
```

```
・屋外歩行
　独歩　FIM7　2>6>7>11>4>1
　途中6番地点のベンチで休憩
```
